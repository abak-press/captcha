captcha
=======

An Rmagick based, Google-style captcha generator for Rails.


Install
-------

	script/plugin install git://github.com/winton/captcha.git

### Generate config/captcha.rb

	rake captcha

### Generate captchas
	
	rake captcha:generate

### Add to .gitignore

	public/images/captchas
